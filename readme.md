# Intro

Commande pour creer notre projet

```
dotnet new console -o 01-helloworld
```
plus tard vous aurez besoin de:
  - dotnet new webapi -o nomDuRepo

Build le projet
```
dotnet build
```

Restore
```
dotnet restore
```

Run le programme (build & run)
```
dotnet run
```

Publier le projet (version de prod)
```
dotnet publish
```
