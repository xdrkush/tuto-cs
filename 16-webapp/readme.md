# Intro

Commande pour creer notre projet

```
dotnet new console -o 01-helloworld
```

Build le projet
```
dotnet build
```

Restore
```
dotnet restore
```

Run le programme (build & run)
```
dotnet watch run
```

Publier le projet (version de prod)
```
dotnet publish
```
